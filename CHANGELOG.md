# Changelog

All notable changes to this project will be documented in this file.

## [0.2.1] - 2026-07-05

### Changed

- Updated package and prompt metadata from `v0.2.0` to `v0.2.1`.
- Renamed the public status marker to `public_skill_v0_2_1` to match the released package version.
- Repacked local distribution zip files so they include the latest routing rules and `test-prompts.json`.

## [0.2.0] - 2026-07-05

### Changed

- Promoted `SKILL.md` from `v0.1.0` draft metadata to `v0.2.0` public release metadata.
- Aligned repository release messaging across `README.md` and `README_EN.md`.

### Added

- Added `CONTRIBUTING.md` with contribution scope, safety boundaries, and PR guidance.

## [0.1.2] - 2026-07-05

### Added

- Added `README_EN.md` for English-speaking GitHub readers.
- Expanded `examples.json` to 10 examples with broader topic coverage.
- Added cross-language README linking and clearer example coverage notes.

## [0.1.1] - 2026-07-05

### Added

- Added `CHANGELOG.md` for release tracking.
- Expanded `examples.json` with more real-world usage cases.
- Expanded `README.md` with scenario-oriented example coverage and release references.

## [0.1.0] - 2026-07-05

### Added

- Initial public draft of the SISFIT coursepack skill.
- Core `SKILL.md` prompt for generating health-education coursepack structures.
- `examples.json` with baseline normal and high-risk examples.
- `boundary_checklist.md` and `xiaohongshu_post.md`.
- MIT license and GitHub-oriented README.
