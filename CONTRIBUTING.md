# Contributing

感谢你对 `sisfit-coursepack` 的改进兴趣。

这个仓库的目标不是输出更多“看起来专业”的健康话术，而是提供一套边界清晰、可直接用于课程策划的身体教育课包骨架。提交修改时，请优先维护这个目标。

## 欢迎提交的内容

- 更清楚的触发描述和调用场景
- 更完整的 `examples.json` 示例覆盖
- 更清晰的 README / README_EN 文档
- 更严格的边界表达示例
- 拼写、格式、链接或结构修正

## 不建议提交的内容

- 医学诊断或疾病判断
- 治疗建议或康复处方
- 对疼痛原因的定论
- “根治”“修复”“治好”“矫正”等治疗化承诺
- 恐吓式转化文案
- 脱离身体教育边界的个体化建议

## 提交前检查

在提交前，请至少确认以下事项：

1. 改动仍然服务于“健康教育课程结构设计”，而不是医学判断。
2. 高风险输入仍然会先提示线下就医或咨询合格专业人员。
3. 没有新增治疗化、诊断化、绝对化表达。
4. `examples.json` 仍然是有效 JSON。
5. README 与 `SKILL.md` 的版本说明没有互相冲突。

## 示例贡献建议

如果你要扩展 `examples.json`，优先补这几类：

- 新主题：肩颈、腰背、髋部、足部、呼吸、睡眠
- 新场景：线下课、线上课、直播课、销售页、朋友圈、小红书、公众号
- 新人群：久坐人群、40+ 女性、孕产期人群、久站工作人群、初学者
- 高风险分流：外伤、肿胀、麻木、夜间痛、活动受限

## 文风要求

- 语言清楚、克制、少术语
- 不制造身体焦虑
- 不夸大效果
- 不替用户判断疾病
- 优先帮助老师建立结构、节奏和表达边界

## Pull Request 建议

PR 描述里建议写清楚：

- 你改了什么
- 为什么要改
- 这次改动影响的是哪一类用户或场景
- 你如何确认没有越过健康教育边界
