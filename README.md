# SISFIT Coursepack Skill

一个面向身体教育老师的中文 Skill，用于把身体主题整理成边界清晰、可交付、可销售的主题课包骨架。

## 它解决什么问题

很多老师已经有主题、动作素材和用户痛点，但缺少一套稳定的输出框架，把这些内容整理成：

- 主题定位
- 适合 / 不适合人群
- 课程目标
- 课程结构与流程
- 课堂口令方向
- 表达边界清单
- 预热内容方向
- 销售页结构

这个 Skill 的目标不是给医学判断，而是帮助老师完成健康教育内容策划和课包产品化表达。

## 适用场景

- 线下主题课设计
- 线上课或直播课结构整理
- 小红书 / 朋友圈 / 公众号预热内容准备
- 课包销售页骨架生成
- 健康教育类课程的风险表达降火

## 不适用场景

- 医学诊断
- 疾病判断
- 疼痛原因定论
- 治疗建议
- 康复处方
- 个体化医疗建议

如果输入涉及外伤、明显肿胀、麻木、发热、夜间痛、活动明显受限等情况，Skill 应先提示线下就医或咨询合格专业人员。

## 安装

将本目录放入本地技能目录：

```text
~/.trae/skills/sisfit-coursepack/
```

目录中至少应包含：

- `SKILL.md`
- `README.md`
- `examples.json`
- `boundary_checklist.md`
- `xiaohongshu_post.md`

## 调用方式

当你要做以下任务时调用这个 Skill：

- 生成一个身体主题课包骨架
- 把身体主题整理成可卖的课程结构
- 输出主题课流程、口令和表达边界
- 生成这个主题的小红书预热方向或销售页骨架

可直接使用类似表达：

- “帮我做一个身体主题课包骨架”
- “把这个主题整理成 60 分钟线下课结构”
- “帮我输出这个主题的课堂口令和销售页结构”

## 输入模板

```text
老师类型：
目标人群：
身体主题：
课程时长：
使用场景：
我想传递的核心观点：
学员最常问的问题：
已有动作或练习素材：
是否涉及特殊人群：
```

## 输出内容

Skill 默认输出以下 12 个部分：

1. 主题定位
2. 适合人群
3. 不适合人群
4. 用户常见痛点语言
5. 课程目标
6. 课程结构
7. 课程流程
8. 课堂口令方向
9. 表达边界清单
10. 预热内容方向
11. 销售页结构
12. 风险提示

## 示例

示例输入见 [examples.json](./examples.json)，其中包含：

- 外伤后膝盖肿胀的高风险边界测试
